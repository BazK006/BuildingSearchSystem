# Building Search System Project

ระบบค้นหาอาคารและแผนที่ภายในโรงเรียนพิบูลมังสาหาร  
พัฒนาเพื่อช่วยให้นักเรียน บุคลากร และผู้มาติดต่อ  
สามารถค้นหาอาคารและตำแหน่งภายในโรงเรียนได้อย่างสะดวกและรวดเร็ว

---

## Features

- ระบบค้นหาอาคารภายในโรงเรียน
- แสดงแผนที่โรงเรียน และเชื่อม Google Maps
- แสดงอาคารทั้งหมดในรูปแบบการ์ด พร้อมดูรูปขยาย
- รองรับการแสดงผลบนมือถือ แท็บเล็ต และเดสก์ท็อป (Responsive)

---

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
